# View Distance Uncapper

Uncaps the game view-distance setting beyond the vanilla menu limits.

## Install

1. Install MelonLoader for the IL2CPP Windows version of *Vietnam War*.
2. Download the release ZIP for this mod.
3. Extract it into your game folder, usually SteamLibrary\steamapps\common\VietnamWar.
4. Confirm that Mods\ViewDistanceUncapper.dll exists in the game folder.
5. Launch the game.

To uninstall, delete Mods\ViewDistanceUncapper.dll from the game folder.

See CONFIGURATION.md.

## Contents

This repository intentionally ships only the compiled mod DLL and install/config documentation. It does not include game files, game assets, MelonLoader, decompiled source, or mod source code.

## Compatibility

Built for the current local Steam/MelonLoader IL2CPP setup used by the author. Other game versions may need a rebuilt DLL.
