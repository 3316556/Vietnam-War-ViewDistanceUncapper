# Install View Distance Uncapper

This is a drag-and-drop MelonLoader mod.

## Player Install

1. Make sure MelonLoader 0.7.2 (Open-Beta) is installed for *Vietnam War* (Unity 2020.3.30f1, Game Version 0.1).
2. Open the game folder: SteamLibrary\steamapps\common\VietnamWar.
3. Drag the included Mods folder into the game folder.
4. Let Windows merge folders if prompted.
5. Start the game.

Correct result:

~~~text
VietnamWar\Mods\ViewDistanceUncapper.dll
~~~

## Uninstall

Delete this file:

~~~text
VietnamWar\Mods\ViewDistanceUncapper.dll
~~~

Do not delete UserData\MelonPreferences.cfg unless you intentionally want to reset settings for all MelonLoader mods.
